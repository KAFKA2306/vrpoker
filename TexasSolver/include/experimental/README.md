# Experimental ディレクトリ

## 概要

本ディレクトリには、実験的な機能や開発中の新機能が格納されています。これらの機能は安定性が保証されておらず、将来のバージョンで変更または削除される可能性があります。

## 注意事項

**警告:**
- 本ディレクトリの機能は実験段階です
- プロダクション環境での使用は推奨されません
- APIは予告なく変更される可能性があります
- ドキュメントが不完全な場合があります

## 使用する場合

実験的機能を使用する場合は、以下の点に注意してください:

1. **テスト環境での評価**
   - 本番環境での使用前に十分なテスト
   - バックアップの取得

2. **フィードバックの提供**
   - 問題を発見した場合はIssueを作成
   - 改善提案を歓迎

3. **バージョン管理**
   - 実験的機能の使用を記録
   - メインブランチへの統合時期を確認
