# UI ディレクトリ

## 概要

本ディレクトリには、TexasSolverのグラフィカルユーザーインターフェース（GUI）に関連するヘッダーファイルが格納されています。Qt 5フレームワークを使用して実装されています。

## 主要機能

### メインウィンドウ
ソルバーのメイン操作画面を提供します。

**機能:**
- ゲームツリー設定
- レンジ入力
- ソルバー実行制御
- 結果表示

### レンジセレクタ
プレイヤーのハンドレンジを視覚的に選択します。

**機能:**
- グリッド表示でのハンド選択
- レンジ文字列の入力/出力
- プリセットレンジの読み込み

### ボードセレクタ
ボードカードを選択します。

**機能:**
- カードの視覚的選択
- 有効なボードのバリデーション
- ランダムボード生成

### 戦略エクスプローラ
計算された戦略を探索します。

**機能:**
- ゲームツリーのナビゲーション
- 各ノードでの戦略表示
- 期待値の可視化
- ヒートマップ表示

### 設定エディタ
ソルバー設定を編集します。

**機能:**
- スレッド数設定
- 精度設定
- ベットサイズ設定
- 高度なオプション

## Qt UIコンポーネント

### カスタムウィジェット
- `QsTreeView`: カスタムツリービュー
- `QsTextEdit`: カスタムテキストエディット
- `HtmlTableView`: HTMLテーブル表示

### ダイアログ
- レンジ入力ダイアログ
- ボード選択ダイアログ
- 設定ダイアログ
- 進捗ダイアログ

## 使用技術

### Qt Framework
- バージョン: 5.12.0以上
- モジュール: Core, Gui, Widgets

### シグナル/スロット
非同期イベント処理を実現:
```cpp
connect(solver, &Solver::iterationCompleted, 
        this, &MainWindow::onIterationCompleted);
```

### スレッド処理
UIスレッドとワーカースレッドの分離:
```cpp
QThread* workerThread = new QThread;
solver->moveToThread(workerThread);
workerThread->start();
```

## ビルド要件

### 必須ライブラリ
- Qt 5.12.0以上
- C++17対応コンパイラ

### オプション
- QtCharts（グラフ表示用）
- QtWebEngine（HTMLレンダリング用）
