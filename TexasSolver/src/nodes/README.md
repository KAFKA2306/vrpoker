# Nodes ディレクトリ（実装）

## 概要

本ディレクトリには、`include/nodes/`で定義されたゲームツリーノードクラスの実装が格納されています。

## ファイル

### GameTreeNode.cpp
基底ノードクラスの実装です。

**実装される機能:**
- ノード深さの管理
- 親子関係の設定
- 共通ユーティリティ関数

### ActionNode.cpp
アクションノードの実装です。

**実装される機能:**
- 利用可能なアクションのリスト管理
- 子ノードの追加と削除
- トレーニング可能オブジェクトの初期化
- 戦略の取得と設定

### ChanceNode.cpp
チャンスノードの実装です。

**実装される機能:**
- 配布可能カードの列挙
- 各カード配布に対する子ノード生成
- 期待値の計算

### ShowdownNode.cpp
ショーダウンノードの実装です。

**実装される機能:**
- ハンド強度の比較
- ポット分配の計算
- 各プレイヤーの期待値算出

### TerminalNode.cpp
終端ノードの実装です。

**実装される機能:**
- 即座のペイオフ計算
- フォールド時の処理
- オールイン時の処理

### GameActions.cpp
ゲームアクション関連の実装です。

**実装される機能:**
- アクション文字列のパース
- アクション種類の判定
- アクション有効性の検証
